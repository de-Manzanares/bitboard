◆ coordinateToIndex()

int coordinateToIndex ( const string &  coordinate)

Converts chess algebraic notation to index notation. To coincide with the bitboards in the ChessBoard, the index of the square is defined as it's position in the 64 bit bitboard.
The index of the square is defined as [0:7] = [h1:a1], [8:15] = [h2:a2] ... [56:63] = [h8:a8].
Starting from 0 at h1, each rank increases the index by 8, and each file increases the index by 1.

Parameters
coordinateThe algebraic notation of the square.
Returns
The index of the square as defined by it's position in the 64 bit bitboard.
