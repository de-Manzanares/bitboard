void fenCommand ( stringstream &  ss,
ChessBoard &  board

Handles the fen command.

Command — fen in — calls the function fenDecode() which updates the board from a FEN string.
Command — fen out — calls the function fenEncode() which prints a FEN string of the current board.

Postcondition
In the case of — fen in — , the board is updated.
Parameters
ssThe stringstream containing the command.
boardThe board to be updated or read from.
